# Learning Godot Projects

This is a repo showing some projects made in Godot while learning about game development concepts and learning to use the game engine. For the near future I will be focusing on 2D games. For creating assets I will be using Aseprite mostly and an Aseprite Wizard plugin to import them into Godot.

## Requirements

- Godot v4.6.1+
- Aesprite

## Commands

- Update all git submodules

    ```bash
    git submodule update --recursive --remote
    ```
